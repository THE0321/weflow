## 💡 About Project

**We Flow**는 업무의 맥락(Context)을 이해하는 AI를 통해 개인의 몰입과 팀의 연결을 돕는 프로젝트입니다.  
복잡한 절차 없이 로그인과 동시에 내 업무에 착수할 수 있는 **'Zero-Friction'** 환경을 제공하며, 수많은 대화 속에서 핵심만 요약해주는 AI 어시스턴트가 당신의 시간을 절약해 드립니다.

<br>

## ✨ Key Features

### 1. 📊 Smart Dashboard : 동기부여와 가시성
> 데이터를 시각화하여 프로젝트의 현재와 나의 위치를 직관적으로 보여줍니다.

<details>
<summary><b>📋 상세 기능 보기 (Click)</b></summary>
  
- **Objective Visualization**: 목표 및 체크리스트 수행 현황을 동적 차트로 구현
- **Progress Insight**: 프로젝트 진행률을 시각적으로 인지하여 즉각적인 현황 파악
- **Gamification**: 목표 달성률에 따른 시각적 피드백으로 업무 동기부여 강화

</details>

<br>

### 2. ✅ Seamless Workflow : 끊김 없는 업무 경험
> 검색은 줄이고, 실행은 빠르게. 개인화된 알고리즘이 나에게 필요한 업무만 보여줍니다.

<details>
<summary><b>📋 상세 기능 보기 (Click)</b></summary>

- **Hyper-Personalization**: 로그인 ID 기반 데이터 필터링으로 "나의 목표"만 간결하게 리스팅
- **Zero-Search Access**: 화면 진입 시 검색 없이 즉시 업무 착수 가능한 UX 환경 구축
- **Accessibility**: 개인별 목표/체크리스트 최적 배치로 업무 접근성 극대화
- **Scalable UI**: 데이터가 늘어나도 혼란스럽지 않은 효율적인 정보 구조 설계

</details>

<br>

### 3. 💬 AI Communication : 핵심을 꿰뚫는 소통
> 메시지를 요약해서 확인하세요.

<details>
<summary><b>📋 상세 기능 보기 (Click)</b></summary>

- **Context Analysis**: AI가 긴 대화의 흐름을 분석하여 문맥 파악
- **Auto Summary**: 핵심 논의 사항 및 결정 내용(Decision Making) 자동 요약
- **Quick Catch-up**: 전체 대화를 읽지 않아도 업무 흐름 파악 가능
- **Instant Collab**: 신규 참여자도 단 몇 초 만에 논의에 합류할 수 있는 브리핑 지원

</details>

<br>

> 단순 반복 분석은 AI에게, 당신은 의사결정에 집중하세요.

<details>
<summary><b>📋 상세 기능 보기 (Click)</b></summary>
  
- **AI Integration**: 채팅, 대시보드, 목표 관리 등 전 영역에 AI 기술 통합
- **Proactive Analysis**: 사용자가 요청하기 전 데이터를 가공하고 요약하여 제공
- **Automation**: 반복 작업 시간 단축 및 업무 자동화 프로세스 구축

</details>
